For all files that start with any one character and are followed by ".txt" do the following: Remove from the first line to pattern1, all lines. Remove from the pattern2 to the last line, all lines.

for f in ?.txt; do ed -s $f <<< $'1,/pattern1/d\nw' ; ed -s $f <<< $'/pattern2/,$d\nw'; done
