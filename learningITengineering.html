2018-11-22 Usefull questions when searching for an error. What ist the system doing when it breaks? Which componentes are responsible? Who is communicating with whom? Is the communication looking as expected? Do the commands get acknowledged? If yes, are the related functions beeing called? Reading code is not the first, but the final step, when figuring out what is awry.
